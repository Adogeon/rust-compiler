# Monkey Lang Compiler In Rust

A Rust implementation of the **Monkey programming language** interpreter from Thorsten Ball's sequel book ["Writing A Compiler In Go"](https://compilerbook.com/). This project explores compiler design implementation using Rust's powerful type system and memory safety guarantees.
